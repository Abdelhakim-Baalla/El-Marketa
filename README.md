# El Marketa

Une plateforme e-commerce backend scalable avec NestJS, TypeScript, Prisma et PostgreSQL.

## Description

Backend pour gestion auth, catalogue, inventaire et commandes. Inspir√© de la culture marocaine avec un vibe souk moderne.

## Installation

1. Clone le repo : `git clone https://github.com/Abdelhakim-Baalla/El-Marketa.git`
2. Installe : `npm install`
3. Configure .env avec DATABASE_URL
4. Migre DB : `npx prisma migrate dev`
5. Lance : `npm run start:dev`
